This is the Cyrus SASL API implentation. It can be used on the client or server side to provide authentication and authorization services. See RFC 4422 for more information.

The latest version is available at: ftp://ftp.andrew.cmu.edu/pub/cyrus-mail

There's a mailing list for Cyrus SASL. Subscribe by sending a message to majordomo@lists.andrew.cmu.edu with the body "subscribe cyrus-sasl". The mailing list is available via anonymous IMAP at imap://cyrus.andrew.cmu.edu/archive.cyrus-sasl or via the web at http://asg.web.cmu.edu/archive/mailbox.php3?mailbox=archive.cyrus-sasl.

If you are looking to port SASLv1 applications to SASLv2, please see doc/appconvert.html

Bugs can be searched/reported at: http://bugzilla.cyrussasl.org
